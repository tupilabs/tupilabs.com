--- title: 'Add tags to Jenkins builds or export specific builds' author: kinow tags: - ideas - jenkins time: '01:20:00' category: blog layout: blog --- This idea is from [@abayer](http://twitter.com/abayer), from these following tweets: ![Tweet 1]({{assets.tweet_001}}) And: ![Tweet 2]({{assets.tweet_002}}) We could simply add tags to builds, and then create some place that would list the builds (or other things) that are tagged appropriately. Or we could add a way to define whicih builds should be exported to some tool, and then via the plug-in API maybe, post to this external tool.
