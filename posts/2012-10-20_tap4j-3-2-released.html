title: 'tap4j 3.2 released' id: 853 author: kinow tags: - tap - tap4j - blog time: '17:32:51' category: news layout: blog ---

tap4j 3.2 is out. This release includes only minor code improvements and the update of TestNG and JUnit dependencies - they are used for the integration extensions. Also, the @author tags were removed, and the code is now formatted similar to the Apache Commons style guide.

If you are using a Maven project, you can add the following to your pom.xml to use tap4j.

<dependency>
  <groupid>org.tap4j</groupid>
  <artifactid>tap4j</artifactid>
  <version>3.2</version>
</dependency>

Find more about tap4j on http://www.tap4j.org
