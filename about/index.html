About

Open Source and Innovation are the core of TupiLabs. We craft code for Open Source projects and also create new Open Source projects. Our contributions can be found in several Open Source projects, as well as in software tailored for some companies.

If you have an idea and need help to interact with an Open Source project, or if you are adopting an Open Source project or technology in your company, feel free to drop us a message.

We can help you:

Fixing issues for new features, enhancements or bugs in Open Source projects
Creating and fostering Open Source projects and communities
Creating courses, training material and documentation
Organizing conferences and meetups
Writing tailored solutions that require good understanding of Open Source tools and technologies
