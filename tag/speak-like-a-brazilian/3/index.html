Search Speak Like A Brazilian using DuckDuckGo

Oct 20, 2012 in speak-like-a-brazilian, duckduckgo | blog

DuckDuckGo offers an awesome service that lets you search for items in Wikias, newspapers or other web sites using a Bang. Basically, you enter ! followed by the bang name and your search query. For example: !answers when do the narwhals bacon?.

And then DuckDuckGo will redirect you to the results on the specific web site. And guess what? Now you can search Speak Like A Brazilian using the !slbr bang in DuckDuckGo. Wanna see how it works? Go to www.duckduckgo.com and search for !slbr cara
