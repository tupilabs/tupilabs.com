Search Speak Like A Brazilian using DuckDuckGo

Oct 20, 2012 in speak-like-a-brazilian, duckduckgo | blog

DuckDuckGo offers an awesome service that lets you search for items in Wikias, newspapers or other web sites using a Bang. Basically, you enter ! followed by the bang name and your search query. For example: !answers when do the narwhals bacon?.

And then DuckDuckGo will redirect you to the results on the specific web site. And guess what? Now you can search Speak Like A Brazilian using the !slbr bang in DuckDuckGo. Wanna see how it works? Go to www.duckduckgo.com and search for !slbr cara

Bom divertimento :D

Oct 20, 2012

title: ‘tap4j 3.2 released’ id: 853 author: kinow tags: - tap - tap4j - news time: ‘17:32:51’ category: blog layout: blog

tap4j 3.2 is out. This release includes only minor code improvements and the update of TestNG and JUnit dependencies - they are used for the integration extensions. Also, the @author tags were removed, and the code is now formatted similar to the Apache Commons style guide.

If you are using a Maven project, you can add the following to your pom.xml to use tap4j.

<dependency>
  <groupid>org.tap4j</groupid>
  <artifactid>tap4j</artifactid>
  <version>3.2</version>
</dependency>

Find more about tap4j on http://www.tap4j.org
