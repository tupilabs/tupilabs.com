Welcome to the Lab!

Here you can find more about our work in progress, and learn more about how we craft code from ideas. Suggestions are more than welcome. Some of the code that will be published in this space is not ready for production.

paskuale
Web spell, grammar, semantic and rules checker. Technologies used: LanguageTool, OpenNLP, jazzy.
CJAN.org
A community to share Java test results in different platforms, locales and time zones.
tupilabs/nebular
Pure Java Fuzzy Logic API, influenced by Matlab, Octave and fuzzylite.
tupilabs/jira_attachments
Project to display the attachments of a JIRA issue.
tupilabs/patches_welcome
Aggregator of issues welcoming patches
newsdiffs
Create server with newsdiffs for Brazil and New Zealand.
Jenkins Seu-Madruga Plug-in
A plug-in that displays Seu Madruga in the build pages.
