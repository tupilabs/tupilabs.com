Garagem Vaga is a web site where people can offer and rent parking lots. It is a free service for all Brazilians. The web site is entirely in Portuguese, and it uses Google Maps for helping the users to find garages.

Speak Like A Brazilian

http://www.speaklikeabrazilian.com

Brazil is one of the largest countries in the world, and due to its diversity (cultural and people) there is a plethora of different ways of saying the same thing. Speak Like A Brazilian was created to help gringos coming to Brazil that would like to use or understand some local expression.

Paskuale

http://www.paskuale.com

Paskuale is a web application that crawls web sites, spell and grammar check them and creates an index score. Users can also submit their own web sites for the crawling and be notified upon spell and grammar errors detected.

BioUno

http://www.biouno.org

BioUno is a project that applies Continuous Integration techniques and tools in Bioinformatics. The main tool used in the project is Jenkins build server.

CJAN.org

http://www.cjan.org

CJAN.org was inspired by CPAN, the Comprehensive Perl Archive Network. You can register Java modules (eg: Apache Commons Functor, Apache Tomcat, etc), read articles and blogs, and upload test results in different environments.

TupiLabs Services

We love creating useful products. And we have a broad range of services to help you to craft an awesome solution too! Wanna see some code? No problem, check our GitHub repositories.

Software development

At TupiLabs we write software in several programming languages (Java, PHP, Python, Lua, Perl), using different tools and techniques. You can contact us for preparing an agile development environment (with Jenkins, Sonar, git, Maven), planning your code versioning structure or for writing software. We have experience working in distributed teams with different time zones and with different cultures and technologies. Our expertise includes both backend and frontend software, varying from CRM and web sites, to Hadoop clusters processing big data and VAS middlewares in mobile carriers. We expect to be challenged to craft fun and interesting solutions that will be helpful to many people.

Software quality

During different cycles of your project you may be questioned about the code quality. Or you can question yourself about the quality of the code written by your team. Our experience in software quality includes code analysis, test automation and testing techniques. TupiLabs can work with you preparing an environment for software quality, training your team or writing manual and automated tests. We contribute to TestLink and we created Jenkins TestLink Plug-in, a plug-in for Jenkins that integrates TestLink and Jenkins. This way you manage your tests in one single tool. We can help you setting up your environment for this integration or with customization.

Continuous integration

Having to work with different programming languages at TupiLabs permit us to explore the diversity of existing ways to build software. Our preference is Jenkins build server, but we work with other servers too. Our services for continuous integration include training, environment set up, integration with other tools and writing plug-ins and scripts. Is your build taking hours to compile a simple web application? Are you having to create more and more nodes to be able to build your software? Having problems finding a perfect set up for your software? While it means headache for the majority, it means fun to us!

Translation

Having your software translated to different languages can attract more users and give a better impression of the software maturity. TupiLabs is a Brazilian company, and Brazilian Portuguese is our principal language. However, all the software that we create is written in English. We contribute to Open Source software translating the software and documentation to/from Brazilian Portuguese, English and Spanish. We have contacts that can help with translation to Japanese, German and French too. It is important to notice that for official documents you will need a certified translator. But for games, software and documentation, feel free to drop us a message.

Open Source

TupiLabs can partner with your team and work on existing issues in Open Source libraries or tools used at the software you are building. If you don't feel confident enough to choose an Open Source solution, TupiLabs can help by creating patches for your bugs, writing new features, writing documentation or training. We also provide an exclusive service of project assessment, where we can quickly delivery a report with an assessment of your project. This assessment includes topics such as community, documentation, ease of code maintainability or strictness to Open Source principles. Take a look at the projects created and contributed to by Tupis.
