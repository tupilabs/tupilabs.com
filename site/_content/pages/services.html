--- title: Services id: 11 author: kinow tags: { } date: '2012-02-04' time: '00:14:55' --- We love creating useful products. And we have a broad range of services to help you to craft an awesome solution too! Take a look at some of the services provided by TupiLabs.

Software development

At TupiLabs we write software in several programming languages (Java, PHP, Python, Lua, Perl), using different tools and techniques. You can contact us for preparing an agile development environment (with Jenkins, Sonar, git, Maven), planning your code versioning structure or for writing software. We have experience working in distributed teamsdifferent time zones and with different cultures and technologies. Our expertise includes both backend and frontend software, varying from CRM and web sites, to Hadoop clusters processing big data and VAS middlewares in mobile carriers. We expect to be challenged to craft fun and interesting solutions, that will be helpful to many people.

Software quality

During different cycles of your project, you may be questioned about the code quality. Or you can question yourself about the quality of the code wrote by you and your team. We have large experience in software quality, including code analysis and testing. TupiLabs can work together with you preparing an environment for software quality, training your team or writing manual and automated tests. We contribute to TestLink and we created Jenkins TestLink Plug-in, a plug-in for Jenkins that lets you integrate both TestLink and Jenkins. This way you manage your tests in one single tool. We can help you setting up your environment for this integration or with customization.

Continuous integration

Having to work with different programming languages at TupiLabs permit us to explore the diversity of existing ways to build software. Our preference is Jenkins build server, but we work with other servers too. Our services for continuous integration include training, environment set up, integration with other tools and writing plug-ins and scripts. Is your building taking hours to compile a simple web application? Are you having to create more and more nodes to be able to build your software? Having problems finding a perfect set up for your software? While it means headache for majority, this is where we have fun, solving problems like these and helping you to build your software better.

Translation

Having your software translated to different languages can attract more users and give a better impression of the software maturity. TupiLabs is a Brazilian company, and Brazilian Portuguese is our principal language. However, all the software that we create is written in English. We contribute to Open Source software translating the software and documentation to/from Brazilian Portuguese, English and Spanish. We have contacts that can help with translation to Japanese, German and French too. It is important to notice that for official documents you will need a certified translator. But for games, software and documentation, feel free to drop us a message.

Open Source

TupiLabs can partner with your team and work on existing issues in Open Source libraries or tools used at the software you are building. If you don't feel confident enough to choose an Open Source solution, TupiLabs can help by creating patches for your bugs, writing new features, writing documentation or training. We also provide an exclusive service of project assessment, where we can quickly delivery a report with an assessment of your project. This assessment includes topics such as community, documentation, ease of code maintainability or strictness to Open Source principles. Have a look at the projects created and contributed to by Tupis.

Illustrations and graphics

At TupiLabs we create most of the graphical material used in our products. This material includes logos, icons, vectorial images and illustrations for articles published in IT magazines. We can create illustrations and graphics for your products or magazine articles.
