Creating software is not only writing code, but also writing documentation and providing help to end users and developers. Here are some books that we have written for our Open Source software. All the books are Open Source as well, so feel free to send contributions or comments too.

Using TAP with Java

This book is an Open Source project intended to give users a further understanding of what is TAP (Test Anything Protocol) and how to use it with Java. In this book examples using TestNG and JUnit are used to better explain how you can use TAP in a Java environment.

  • Project: tap4j - a TAP implementation in Java
  • HTML: http://www.tap4j.org/book
  • PDF: http://www.tap4j.org/book/book.pdf
  • Sources: https://github.com/kinow/using-tap-with-java

Jenkins TestLink Plug-in Tutorial

The Jenkins TestLink Plug-in Tutorial is intended to provide a better understanding on how to use the plug-in to integrate Jenkins and TestLink. It is an Open Source project, so contributions and suggestions are welcome.

  • Project: Jenkins TestLink Plug-in
  • HTML: http://tupilabs.com/books/jenkins-testlink-plugin-tutorial/en/book.html
  • PDF: http://tupilabs.com/books/jenkins-testlink-plugin-tutorial/en/book.pdf
  • Sources: https://github.com/tupilabs/jenkins-testlink-plugin-tutorial
