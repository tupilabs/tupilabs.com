Open source is great! You can meet new people contributing or creating open source solutions. And you learn a lot too! TupiLabs has contributed to Open Source community with a large number of projects, and has plans to contribute with many other ideas, patches, participating in discussions, translating and updating documentation, and having a lot of fun! Check out some of our contributions to the Open Source community.

Started by Tupis

Bellman-Zadeh Fuzzy Decision Maker

https://sourceforge.net/projects/bellmanzadeh/

A tool based on the model proposed by Bellman & Zadeh (1970) fuzzy logic (fuzzy sets) decision support paper. The first version of the application was developed in Java (pure java + jgoodies and some other APIs).


BioUno

http://www.biouno.org/

BioUno is a biology workflow system built on top of Jenkins and it uses biology tools like MrBayes, FigTree, Beast, Bucky among others.


GIFFrame

https://sourceforge.net/projects/gifframe/

GifFrame is a simple java program that, given a animated gif image as input, it extracts its frames to a defined output path, or the reverse. Given a directory with multiple gifs, it joins them into a single animated gif.


GTest TAP Listener

GTest TAP Listener is an event listener for GoogleTest C++ Testing Framework that enables tests to output TAP.


Jenkins CCM Plug-in

https://wiki.jenkins-ci.org/display/JENKINS/CCM+Plugin

This plug-in generates reports on cyclomatic complexity for .NET code using CCM.


Jenkins Seu Madruga Plug-in

https://github.com/tupilabs/seumadruga-plugin

This plug-in is a copy of Chuck Norris plug-in, and replaces the butler by Seu Madruga photo, and also include some of his famous phrases.


Jenkins TAP Plug-in

https://wiki.jenkins-ci.org/display/JENKINS/TAP+Plugin

This plug-in adds support to TAP test result files to Jenkins. It lets you specify an ant-like pattern for a directory that contains your TAP files.


Jenkins TestLink Plug-in

https://wiki.jenkins-ci.org/display/JENKINS/TestLink+Plugin

This plug-in integrates Jenkins and TestLink and generates reports on automated test execution. With this plug-in you can manage your tests in TestLink, schedule and control in Jenkins, and execute using your favorite test execution tool (TestPartner, Selenium, TestNG, Perl modules, PHPUnit, among others). Nice uhn?


Maven Jacobe Plugin

https://sourceforge.net/projects/maven-jacobe/

This is a Maven plug-in that calls TIOBE's Jacobe code beautifier.


nebular fuzzy logic Java API

http://nebular.sourceforge.net/

A fuzzy API written purely in Java. This will implement the features present in Matlab Fuzzy Toolbox. What is really interesting here, is the use of Apache Commons Functor. With this functional programming API, the end user can write his own functor (that will be a membership function).


tap4j

http://www.tap4j.org/

tap4j is an implementation of the TAP - Test Anything Protocol - for Java. TAP is a standard way to document test execution. There are two main roles in TAP, a producer and a consumer.


TestLink Java API

http://testlinkjavaapi.sourceforge.net/

TestLink Java API is a pure Java API that interfaces TestLink XML RPC API.

Contributed to by Tupis

Apache Commons Functor

http://commons.apache.org/sandbox/functor/

Apache Commons Functor is a functional programming API for Java.


cpew

https://github.com/kremso/cpew

Create new eclipse workspace and copy all settings from the old workspace.


Jenkins

http://jenkins-ci.org/

Jenkins is an award-winning application that monitors executions of repeated jobs, such as building a software project or jobs run by cron.


PHP-Google-Map-Class

https://github.com/ezimuel/PHP-Google-Map-Class

PHP-Google-Map-Class (GMaps) is a light PHP class that uses the Google Maps API to retrieve information from addresses.


TestLink

http://teamst.org/ Test management tool.


testlink-api-java-client

http://code.google.com/p/testlink-api-java-client/

This project is an client that interfaces with the TestLink php API via XML RPC.


testlink-eclipse-plugin

http://code.google.com/p/testlink-eclipse-plugin/

The eclipse plugin interacts with your TestLink website instance and allows the user of the plug-in to automate test recorded in the TestLink application.


TunesViewer

http://tunesviewer.sourceforge.net/

TunesViewer is a small, easy to use program to access itunes-university media and podcasts in Linux.
