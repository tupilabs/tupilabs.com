This is an interesting case that I received via e-mail from G. M. after he read the post . Basically, using an approach similar to the testlink-plugin for Jenkins, one could send the coverage results to TestLink.

TestLink has requirements x test case coverage information, and Sonar, Jenkins, Cobertura and other tools give you source code x test coverage. Crossing these information could be interesting.
