--- title: 'Configurando o Jenkins para usar o Active Directory' author: kinow tags: - jenkins - tutorial - active directory - portuguese time: '15:37:03' category: blog layout: blog --- Depois de ver algumas mensagens na [lista do Jenkins em Português](https://groups.google.com/forum/#!forum/jenkinsci-br) decidir escrever este post para ajudar quem quiser configurar o [Jenkins](http://jenkins-ci.org) para usar o [Microsoft Active Directory](http://en.wikipedia.org/wiki/Active_Directory). O Jenkins j&aacute; possui autenticação e uma base de usuários própria. O que faremos é apenas utilizar o Active Directory (AD) como a base de usuários. Para isso basta instalar o [Jenkins Active Directory plugin](http://wiki.jenkins-ci.org/display/JENKINS/Active+Directory+Plugin) e reiniciar. ![Plug-in do AD]({{assets.ad_jenkins_001}}) Agora precisamos configurar o Jenkins para acessar o Active Directory. Vá até _Gerenciar Jenkins / Configurar Segurança Global_ e em Controle de Acesso, selecione "Active Directory". Entre o domínio e os dados para se conectar no AD. Caso você não saiba os dados do seu AD, pergunte para alguém de infraestrutura ou tente se conectar ao AD com algum cliente de LDAP. ![Configurando o Plug-in]({{assets.ad_jenkins_002}}) Pronto! Agora basta você escolher o modelo de autorização. Na maioria das vezes utilizo a estratégia baseada em matriz por projeto. Se você utilizar a mesma, adicione o nome de usuário de rede e atribua as permissões corretas. Espero que ajude, boa diversão!
